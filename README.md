# Spotify Playlist creator

current use: creates a playlist based on your mode so the user can enjoy and play in his/her spotify app.
Goal: find information of your artist, create the playlist based on your taste and history then send the play list to your spotify account


## Instructions for Build and Use

Steps to build and/or run the software:

1. git clone <repository-url>
2. npm install
3. create .env (in root directory with the spotify API credentials)

Instructions for using the software:

1. npm start
2. select current mood and click generate
3.

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Node.js v18+
* npm v9+
* dependencies:
    * dotenv - management envionment variables
    * prettier - code formatting
    * spotify-web-api-node - interface with the spotify API
